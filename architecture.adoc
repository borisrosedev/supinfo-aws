= Architecture in AWS

== Well-Architected Framework

Il s'agit d'un cadre de conception AWS qui met l'accent sur les meilleures pratiques dans les domaines :
[%step]
* des coûts
* de la performance
* de la sécurité
* de la fiabilité
* de l'efficacité opérationnelle.
